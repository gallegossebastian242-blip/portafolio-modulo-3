Consulta de Clima en Tiempo Real

Funcionalidad desarrollada en JavaScript que consume la API de OpenWeather. Permite consultar el clima actual y un pronóstico de corto plazo representado en tarjetas informativas.

El pronóstico muestra 5 intervalos reales entregados por la API (aprox. cada 3 horas).
